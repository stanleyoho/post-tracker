# Post Tracker

A tool for tracking and downloading posts from various social media platforms.

## Features
- **Twitter/X**: Scrape posts from specific profiles.
- **Threads**: Monitor and extract threads.
- **Data Management**: Saves tracked data locally.

## Setup
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Configure environment variables in `.env`.
3.  Run the application:
    ```bash
    npm start
    ```

## Directory Structure
- `src/`: Source code
- `data/`: Local database/state
- `downloads/`: Downloaded media/content
- `doc/`: Documentation
